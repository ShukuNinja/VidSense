# Screenshots

Drop your captures here with these exact filenames — the main `README.md` references
them, so they'll render automatically once added.

| File | What to capture |
|---|---|
| `demo.gif` | **Hero.** A short (5–10 s) screen recording: open a chat, ask a question, watch the answer stream in with a ⏱ citation. |
| `answer.png` | A chat showing a question, the grounded answer, and a timestamp citation chip. |
| `ingest.png` (or `ingest.gif`) | The "New chat" modal (URL + start/end) and/or the Download → Audio → Transcribe → Chunk → Index progress stepper. |
| `sidebar.png` | The sidebar listing several chats with their status dots. |
| `login.png` | The email/password sign-in screen. |

Tips:
- Capture at a normal window size; PNG for stills, GIF for short flows (keep GIFs small —
  a screen recorder + a GIF export, or tools like ScreenToGif / LICEcap, work well).
- Once added, `git add assets/screenshots/<file>` and commit.
